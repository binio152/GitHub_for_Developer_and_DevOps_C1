Konosuba Anime Front End Website

This is a simple front-end website for the anime "Konosuba: God's Blessing on This Wonderful World!" built using HTML, CSS, and JavaScript. The website features information about the anime, including characters, episodes, and a gallery.

Features
Character Profiles: Learn about the main characters of the anime, including their backgrounds and personalities.
Episode Guide: Get a list of episodes with summaries and air dates.
Gallery: View images and artwork related to the anime.
Responsive Design: The website is designed to be responsive and works well on both desktop and mobile devices.
